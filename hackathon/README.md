# Hackathon 项目

## 目录说明
此目录用于记录参与 Hackathon 活动的项目和学习成果。

## 文件格式
每个 Hackathon 项目应包含：
- 项目名称
- Hackathon 名称和日期
- 项目描述
- 团队成员
- 技术栈
- 项目成果
- 演示链接
- 反思与收获

## 使用方法
1. 在此目录下创建新文件：`YYYY-MM-DD-Hackathon-项目名称.md`
2. 记录完整的 Hackathon 参与过程
3. 包含项目代码链接和演示视频/链接

## 示例
```
2024-06-15-Web3Hack-去中心化投票系统.md
2024-07-20-AI-Hackathon-智能合约审计工具.md
2024-08-10-ETHGlobal-链上数据可视化.md
```
