# Handbook Feedback

这个目录用于存放学习过程中对 Handbook 的反馈和建议。

## 反馈格式

每条反馈建议包含：

1. **Handbook 页面链接**: 具体哪个页面
2. **问题描述**: 发现的问题或困惑
3. **建议改法**: 如何改进
4. **来源**: 你的学习笔记链接

## 文件命名

- 使用 `YYYY-MM-DD-topic.md` 格式
- 例如: `2026-05-19-llm-intro-typo.md`

## 示例

```markdown
# Handbook 反馈: LLM 入门章节错别字

## 页面链接
https://aiweb3.school/zh/handbook/ai/llm/

## 问题描述
在"什么是 LLM"部分，第三段有错别字："大语言模型是一种**任工**智能技术"
应改为："大语言模型是一种**人工**智能技术"

## 建议改法
修正错别字

## 来源
[学习笔记](../daily/2026-05-19.md)
```
